Feladat
-------

Egy snake programhoz mesterséges intelligencia készítése. Az elkészített AI által vezérelt snake-nek meg kell próbálni legyőzni a mintaként adott AI-t! A túlélés mellett fontos, hogy törekedjen az étel megszerzésére.
A forráskód legyen javadoc-kal dokumentálva!


Mellékletek
-----------

A következő három mappa található a mellékletek között:
- javadoc
- lib
- src

A javadoc könyvtárban található a program dokumentálása. Indítás az index.html fájllal történik. A dokumentáció olvasását ajánlott az Interfaces/Adat interfésszel kezdeni. Fontosabb interfészek, enumok:
Enums/PalyaElemek, Interfaces/Test, Enums/Iranyok, Interfaces/Ai.

A lib könyvtárban a teljes snake programot tartalmazó .jar fájl található.

Az src könyvtárban található Main.java indítja el a Snake programot, két Mintai-t inícializál. A második helyettesítendő az elkészített AI-val.
A Mintai.java osztály egy példa implementáció.
